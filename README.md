# Falla Passeig - Web

Projecte web de la Falla Passeig de Tavernes de la Valldigna.

## Preview Site

Podeu veure una vista prèvia del lloc en: [https://fallapasseig.github.io/web/](https://fallapasseig.github.io/web/)

## Desenvolupament

Aquest projecte està construït amb [Astro](https://astro.build/).

### Començar

```bash
npm install
npm run dev
```

### Construcció

```bash
npm run build
```
